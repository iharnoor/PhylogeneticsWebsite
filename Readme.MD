## Requirements for Python

1. Install the following packages:  
(If pip3 does not work, try pip)
    * install flask_cors  
    ```pip3 install -U flask-cors```
    * install pydot  
    ```pip3 install pydot```
    * Similarly install  
    ```pip3 install pandas```  
    ```pip3 install numpy```  
    ```pip3 install flask```  

2. Check JRE 8 & JDK:  
  To test: try ```java --version ``` or ```java -version ```


## To Run the program
1. cd Server 
2. Run ```python3 Server.py```
3. Open index.html
